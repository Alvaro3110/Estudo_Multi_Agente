# 📊 Santander AI - Visão Geral para o Gestor

## 🎯 Objetivo do Projeto
O **Santander AI** é uma plataforma corporativa avançada desenvolvida para transformar a análise de dados e a tomada de decisão gerencial. Ele substitui a navegação em dashboards complexos e estáticos por uma interface inteligente governada por agentes, capaz de processar linguagem natural, analisar bases de dados extensas e fornecer insights contextualizados em tempo real.

## 🚀 Valor para o Negócio (Business Value)
- **Rapidez na Decisão:** Redução do tempo de extração de relatórios através do Chat Integrado (Perguntar para IA).
- **Precisão Contextual:** Em vez de relatórios genéricos, o sistema gera relatórios dinâmicos e focados para 9 departamentos distintos (Financeiro, Empréstimo, Bacen, etc.).
- **Segurança e Governança:** Integração do fluxo HITL (Human-In-The-Loop) — a inteligência artificial sugere ações críticas, mas a decisão de aprovação, delegação ou rejeição permanece com o humano.
- **Redução de Viéses:** Utilização de um *Anti-Bias Engine* em backend para garantir que as decisões de crédito e avaliações obedeçam às normativas de equidade.

## 🔑 Funcionalidades Principais
1. **Hub Departamental (Etapa 1):** Grid executivo no padrão Santander, listando o status em tempo real das operações de cada unidade de negócio.
2. **Relatório Dinâmico de Agentes (Etapa 2):** Ao entrar em um departamento, 3 agentes especializados emitem laudos sintéticos baseados em KPIs e no banco de dados ativo.
3. **Chat Investigativo (Histórico e Perguntas Livres):** Os gestores podem consultar o "Histórico" das análises ou iniciar uma "Nova Consulta" investigando sub-tópicos de forma conversacional com a IA.
4. **Painel de Ações (Etapa 3):** Caixa de entrada de decisões diárias priorizadas (Alta, Média, Baixa) que exigem intervenção manual inteligente.

## 👥 Para Quem é Essa Ferramenta?
- **Gestores Executivos (C-Level & Diretores):** Para visão top-down através da "Visão Consolidada".
- **Líderes de Operação e Compliance (Mesa de Precificação/Risco):** Para ação rápida nos alertas gerados pelos agentes nas seções departamentais.
