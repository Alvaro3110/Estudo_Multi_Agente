   - Tabela de Clientes: Dados demográficos dos clientes, que podem ser úteis para segmentação.

   - Filtrar os dados por período (por exemplo, últimos 12 meses) para análise de crescimento.

3. **Cálculo das Métricas:**
   - **Volume de Vendas:** Somar o valor total das vendas no período selecionado.
   - **Crescimento Percentual:** Calcular a variação percentual em relação ao período anterior usando a fórmula: 
     \[
     Crescimento \% = \frac{(Vendas\_Atuais - Vendas\_Anteriores)}{Vendas\_Anteriores} \times 100
     \]
   - **Comparação com Períodos Anteriores:** Criar um gráfico ou tabela que mostre as vendas mensais ou trimestrais para facilitar a visualização das tendências.
